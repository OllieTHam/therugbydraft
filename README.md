# 20-0

20-0 is a browser-based rugby draft game built with Vite, React, and Tailwind CSS. Players take turns picking from a pool of real club players to assemble a team, then simulate a match whose outcome is driven by an under-the-hood engine that weighs position ratings, international experience, and team balance — aiming to capture the tactical depth of rugby union in a fast, pick-up-and-play format.
